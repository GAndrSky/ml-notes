🤔 Почему SGD+Momentum недостаточно?

В нейронных сетях разные параметры имеют очень разные масштабы градиентов:

embedding weights: grad ≈ 1e-4 (редкие обновления) attention Q/K/V: grad ≈ 0.01 FF layer: grad ≈ 0.1 output layer: grad ≈ 1.0

При одном глобальном lr: либо embedding почти не обучается, либо output layer взрывается.

💡 Идея адаптивных методов:
Дать каждому параметру θᵢ свой эффективный learning rate:

η_eff(i) = η / √(EMA[g²]_i)

Параметр с историей больших градиентов → маленький шаг.
Параметр с историей маленьких градиентов → большой шаг.

🛡️ AdamW — исправление Weight Decay

Проблема: L2 regularization ≠ Weight Decay в Adam

Adam + L2 (НЕПРАВИЛЬНО): g_t ← ∂L/∂θ + λ·θ ← λθ входит в EMA момента θ ← θ − η · m̂/(√v̂+ε) ← decay масштабируется 1/√v̂ ! Параметры с большим v̂ (частые, шумные признаки): 1/√v̂ → мало → weight decay слабее → недорегуляризированы ❌ AdamW (ПРАВИЛЬНО — Loshchilov & Hutter, 2019): g_t ← ∂L/∂θ ← чистый gradient m_t ← β₁m + (1−β₁)g v_t ← β₂v + (1−β₂)g² θ ← θ − η·m̂/(√v̂+ε) − η·λ·θ ← decay отдельно
💡 Decoupled weight decay обеспечивает:
• Равномерный λ для всех параметров
• Правильная L2-регуляризация в пространстве θ
• Доказано: Adam+L2 ≠ SGD+L2, AdamW+wd ≈ SGD+L2
📌 Практика: weight_decay=0.01..0.1 для большинства Transformer.
НЕ применять к bias и нормировочным параметрам (γ, β в LayerNorm).

🦁 Lion — EvoLved Sign Momentum (Chen et al., 2023)

Найден автоматическим поиском программ (symbolic program search). Проще Adam, работает не хуже или лучше.

Lion (полная формула): update = sign(β₁·m_{t-1} + (1−β₁)·g_t) θ_{t+1} = θ_t − η·update − η·λ·θ_t m_t = β₂·m_{t-1} + (1−β₂)·g_t ← только потом! Гиперпараметры: β₁ = 0.9 (для update) β₂ = 0.99 (для EMA памяти) η = 1e-4 (в 3-10× меньше чем Adam) λ = weight decay (обязателен!)

Ключевые отличия от Adam

Adam: θ ← θ − η · m̂/(√v̂+ε) (масштабированный) Lion: θ ← θ − η · sign(...) (единичный шаг!) Lion делает шаги фиксированного размера η в любом направлении. Нет деления на √v̂ → нет адаптации масштаба. Только направление, не величина.

Интерактивный sign-step

β₁ = 0.90
lr Lion = 0.0003
💡 Почему sign?
sign(v) = v/|v| — единичный вектор в направлении v.
Lion — это SGD с Momentum, но шаг всегда фиксированный η.
Более «демократичен»: все параметры обновляются одинаково сильно.
📌 Практика Lion (2023–2024):
• lr в 3-10× меньше чем у Adam
• Сильный weight decay обязателен (0.1..1.0)
• Меньше памяти: нет v_t (только m_t)
• Хорошо для ViT, LLM:
  Google Brain, Orca 2, некоторые production модели
