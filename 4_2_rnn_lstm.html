1. Рекуррентная нейронная сеть — идея

RNN обрабатывает последовательность x₁, x₂, …, xₜ, поддерживая hidden state hₜ — «память» о прошлом.

hₜ = tanh( Wₕ·hₜ₋₁ + Wₓ·xₜ + b ) yₜ = Wᵧ·hₜ + bᵧ

Параметры Wₕ, Wₓ, b — одни и те же на каждом шаге. Это и есть parameter sharing во времени.

Размерности: xₜ ∈ ℝⁿ, hₜ ∈ ℝᴴ, Wₓ ∈ ℝᴴˣⁿ, Wₕ ∈ ℝᴴˣᴴ, Wᵧ ∈ ℝᴼˣᴴ
Параметров: H·n + H·H + H + O·H + O = H(n + H + 1) + O(H+1)

Типы задач

Many-to-One
x₁..xₜ → y
Классификация текста, сентимент-анализ
Many-to-Many
x₁..xₜ → y₁..yₜ
POS-tagging, NER, seq2seq
One-to-Many
x₁ → y₁..yₜ
Генерация текста, image captioning

2. Проблема затухающего/взрывного градиента

При backprop через T шагов градиент умножается на Wₕ и tanh' на каждом шаге:

∂L/∂h₁ = ∂L/∂hₜ · Π_{k=2}^{T} ( diag(1-hₖ²) · Wₕᵀ )

diag(1-hₖ²) — Якобиан tanh: tanh'(z) = 1 - tanh²(z) ∈ [0,1]

Норма произведения матриц

‖∂hₜ/∂h₁‖ ≤ (λ_max · max|tanh'|)^(T-1)

где λ_max — наибольшее сингулярное число Wₕ.

Затухание

λ_max · max|tanh'| < 1
→ градиент → 0 экспоненциально
→ RNN «забывает» далёкое прошлое
Взрыв

λ_max · max|tanh'| > 1
→ градиент → ∞ экспоненциально
→ обучение взрывается (NaN)
Как бороться со взрывом: Gradient Clipping — обрезаем норму: if ‖g‖ > threshold: g ← g · threshold/‖g‖
Как бороться с затуханием: LSTM/GRU, ResNet-like skip connections, инициализация Wₕ = I

3. LSTM — Long Short-Term Memory (Hochreiter & Schmidhuber, 1997)

LSTM добавляет cell state cₜ — «конвейерная лента» с минимальными трансформациями, управляемая тремя воротами.

Forget Gate fₜ
fₜ = σ(Wf·[hₜ₋₁, xₜ] + bf)
«Что забыть из cell state?» fₜ∈(0,1) поэлементно
Input Gate iₜ
iₜ = σ(Wi·[hₜ₋₁, xₜ] + bi)
c̃ₜ = tanh(Wc·[hₜ₋₁, xₜ] + bc)
«Что записать?» iₜ — сколько, c̃ₜ — что
Output Gate oₜ
oₜ = σ(Wo·[hₜ₋₁, xₜ] + bo)
«Что вывести в hₜ?»

Обновление cell state и hidden state

cₜ = fₜ ⊙ cₜ₋₁ + iₜ ⊙ c̃ₜ hₜ = oₜ ⊙ tanh(cₜ)

⊙ — поэлементное произведение (Hadamard product)

Почему LSTM решает vanishing gradient?
Градиент по cₜ₋₁: ∂cₜ/∂cₜ₋₁ = fₜ — это просто умножение на fₜ, без матричного произведения и tanh'!
При fₜ ≈ 1 градиент течёт практически без затухания через T шагов.

Параметры LSTM

4 · (H·H + H·n + H) = 4H(H + n + 1) параметров

В 4 раза больше, чем у RNN — за каждый из 4 вентилей (f, i, c̃, o)

4. GRU — Gated Recurrent Unit (Cho et al., 2014)

GRU — упрощённая версия LSTM: два ворота вместо трёх, нет отдельного cell state.

Reset Gate rₜ
rₜ = σ(Wr·[hₜ₋₁, xₜ] + br)
«Насколько забыть прошлое при вычислении кандидата?»
Update Gate zₜ
zₜ = σ(Wz·[hₜ₋₁, xₜ] + bz)
«Сколько старого vs нового состояния взять?»
h̃ₜ = tanh(W·[rₜ ⊙ hₜ₋₁, xₜ] + b) ← кандидат hₜ = (1 - zₜ) ⊙ hₜ₋₁ + zₜ ⊙ h̃ₜ ← линейная интерполяция
Интуиция zₜ: при zₜ→0 hₜ≈hₜ₋₁ (игнорируем вход, помним прошлое). При zₜ→1 hₜ≈h̃ₜ (полностью обновляемся).

Сравнение RNN / GRU / LSTM

МодельВоротаПараметрыCell StateСкорость
RNN0H(n+H+1)+O(H+1)нетбыстрее
GRU2 (r, z)3H(n+H+1)+O(H+1)нетсредняя
LSTM3 (f, i, o)4H(n+H+1)+O(H+1)дамедленнее

На практике GRU ≈ LSTM по качеству на большинстве задач, но в 1.3× быстрее.

Параметры GRU

3 · (H·(H+n) + H) = 3H(H + n + 1) параметров

5. Backpropagation Through Time (BPTT)

Развёртываем RNN в глубокую сеть по оси времени и применяем обычный backprop:

L = Σₜ Lₜ(yₜ, ŷₜ) ← суммарный loss по всем шагам

Градиент по Wₕ

∂L/∂Wₕ = Σₜ ∂Lₜ/∂Wₕ ∂Lₜ/∂Wₕ = Σ_{k=1}^{t} ( ∂Lₜ/∂hₜ · Π_{j=k+1}^{t} ∂hⱼ/∂hⱼ₋₁ ) · ∂hₖ/∂Wₕ
Шаг 1.∂hₜ/∂hₜ₋₁ = diag(1-hₜ²) · Wₕᵀ ← Якобиан одного шага
Шаг 2.∂hₜ/∂h₁ = Π_{k=2}^{t} diag(1-hₖ²) · Wₕᵀ ← произведение Якобианов
Шаг 3.∂hₖ/∂Wₕ = hₖ₋₁ᵀ ← вклад на шаге k

Truncated BPTT

Полный BPTT при T=1000 — 1000 шагов назад. На практике: Truncated BPTT — обрезаем backprop до k₂=35-200 шагов, forward pass можно не обрезать.
∂L/∂Wₕ ≈ Σ_{t} Σ_{k=t-k₂}^{t} ( ∂Lₜ/∂hₜ · Π ∂hⱼ/∂hⱼ₋₁ ) · ∂hₖ/∂Wₕ

Gradients для LSTM через время

∂cₜ/∂cₜ₋₁ = fₜ ← только поэлементное умножение, нет Wₕ, нет tanh'!

Это «шоссе градиента» (gradient highway) — ключевое преимущество LSTM над RNN.

Числовой пример: BPTT для RNN (T=3)

Дано:T=3, H=1, Wₕ=0.5, hₜ=tanh(Wₕ·hₜ₋₁+Wₓ·xₜ), h₀=0, x=[1,1,1], L=h₃²
Forward:h₁=tanh(0+Wₓ)≈0.76, h₂=tanh(0.5·0.76+Wₓ)≈0.91, h₃=tanh(0.5·0.91+Wₓ)≈0.95 (Wₓ=1)
∂L/∂h₃= 2·h₃ = 1.90
∂L/∂Wₕ= ∂L/∂h₃ · [(1-h₃²)·h₂ + (1-h₃²)·Wₕ·(1-h₂²)·h₁ + …]

7. PyTorch — реализация

Базовый RNN вручную

import torch, torch.nn as nn class ManualRNN(nn.Module): def __init__(self, input_size, hidden_size, output_size): super().__init__() self.Wx = nn.Linear(input_size, hidden_size, bias=False) self.Wh = nn.Linear(hidden_size, hidden_size, bias=True) self.Wy = nn.Linear(hidden_size, output_size) def forward(self, x, h0=None): B, T, n = x.shape; H = self.Wh.weight.shape[0] h = torch.zeros(B, H) if h0 is None else h0 outs = [] for t in range(T): h = torch.tanh(self.Wx(x[:, t]) + self.Wh(h)) outs.append(self.Wy(h).unsqueeze(1)) return torch.cat(outs, dim=1), h # (B,T,O), (B,H)

nn.RNN / nn.LSTM — стандартный API

rnn = nn.RNN(input_size=10, hidden_size=32, num_layers=2, batch_first=True, dropout=0.2) lstm = nn.LSTM(input_size=10, hidden_size=32, num_layers=2, batch_first=True, dropout=0.2, bidirectional=True) x = torch.randn(8, 20, 10) # (batch=8, seq=20, features=10) out_r, h_r = rnn(x) # out: (8,20,32), h: (2,8,32) out_l, (h_l, c_l) = lstm(x) # out: (8,20,64), h,c: (4,8,32) # # 64 = 32*2 (bidirectional)

Sequence Classification

class SentimentLSTM(nn.Module): def __init__(self, vocab_size, emb_dim=64, hidden=128, n_layers=2): super().__init__() self.emb = nn.Embedding(vocab_size, emb_dim, padding_idx=0) self.lstm = nn.LSTM(emb_dim, hidden, n_layers, batch_first=True, dropout=0.3) self.fc = nn.Linear(hidden, 1) # binary classification def forward(self, x): # x: (B, T) token ids emb = self.emb(x) # (B, T, emb_dim) out, (h, c) = self.lstm(emb) # out: (B,T,H) # Берём hidden state последнего слоя, последнего шага last = h[-1] # (B, H) return self.fc(last).squeeze(-1) # (B,)

LSTM вручную (NumPy)

import numpy as np def sigmoid(x): return 1 / (1 + np.exp(-x)) def lstm_step(x, h_prev, c_prev, W, b): """ x: (n,) h_prev,c_prev: (H,) W: (4H, n+H) b: (4H,) """ combined = np.concatenate([h_prev, x]) # (H+n,) gates = W @ combined + b # (4H,) H = len(h_prev) f = sigmoid(gates[0*H : 1*H]) # forget i = sigmoid(gates[1*H : 2*H]) # input o = sigmoid(gates[3*H : 4*H]) # output c_tilde = np.tanh(gates[2*H : 3*H]) # candidate c = f * c_prev + i * c_tilde # new cell h = o * np.tanh(c) # new hidden return h, c
