7.3 Diffusion Models

Diffusion learns generation as iterative denoising: corrupt real data with controlled noise, then train a model to reverse that corruption step by step.

Prerequisites1.3 Probability, 3.4 Loss, 7.1 VAE
Estimated time70-90 min
Job relevanceMedium for general ML, High for GenAI roles
DepthSolid

TL;DR

A diffusion model does not map random noise to an image in one jump. It learns many small denoising steps, where each step removes a predictable amount of noise from the current sample.

Forward process: destroy data in a controlled way

The forward process is fixed, not learned. At each timestep it adds a small amount of Gaussian noise. After many steps, the sample becomes almost pure noise.

q(x_t | x_{t-1}) = N(x_t; sqrt(1-beta_t) x_{t-1}, beta_t I)
Что означают символы
x_tnoisy sampleData point after t noise steps: image, audio, latent vector or embedding.
x_{t-1}previous sampleSlightly cleaner version from the previous timestep.
beta_tnoise scheduleSmall positive value controlling how much noise is added at step t.
Iidentity covarianceNoise is independent across dimensions in the basic DDPM formulation.
Интуиция: the process slowly washes structure out of the data while keeping the corruption mathematically trackable.
Аналогия: like adding fog to a photo in many tiny steps until only fog remains.
Числовой пример: if beta_t = 0.01, the step keeps about sqrt(0.99) = 0.995 of the previous signal and injects 1% noise variance.
Что это значит на практике
The schedule is a curriculum. Early timesteps teach the model to recover fine details from light corruption. Late timesteps teach it to create global structure from heavy noise.

Closed form: sample any noise level directly

DDPM has a useful closed form: instead of applying every small noising step, we can sample x_t directly from x_0 and one Gaussian noise draw.

x_t = sqrt(alpha_bar_t) x_0 + sqrt(1-alpha_bar_t) epsilon
Что означают символы
x_0clean dataOriginal training example before corruption.
alpha_bar_tremaining signalCumulative product of alpha values; decreases as t grows.
epsilonGaussian noiseRandom noise sampled from a standard normal distribution.
sqrt(...)variance scalingKeeps total variance controlled while signal is replaced by noise.
Интуиция: x_t is a weighted mixture of real signal and random noise.
Аналогия: like blending a clean photo with static using a slider.
Числовой пример: if alpha_bar_t = 0.25, then x_t keeps 0.5 of the clean signal and adds sqrt(0.75) ≈ 0.866 noise scale.

Reverse process: learn to remove noise

The reverse process is learned. A neural network receives a noisy sample and timestep, then predicts how to move one step toward cleaner data.

L_simple = E || epsilon - epsilon_theta(x_t, t) ||^2
Что означают символы
epsilontrue noiseThe exact noise that was added to create x_t during training.
epsilon_thetapredicted noiseNeural network output; theta are trainable weights.
ttimestepTells the network how noisy the input is.
||.||_2^2MSE penaltyLarge mistakes in noise prediction are penalized quadratically.
Интуиция: if the model can identify the noise, it can subtract it and recover structure.
Аналогия: like learning the hiss in an audio recording so you can remove it.
Числовой пример: if true noise is [0.3, -0.1] and prediction is [0.2, 0.0], the squared error is 0.01 + 0.01 = 0.02.

Common mistakes

  • Thinking diffusion directly predicts pixels from pure noise in one step. It predicts local denoising directions.
  • Ignoring the timestep embedding. The same noisy image means different things depending on noise level.
  • Confusing DDPM training with sampling. Training can use random timesteps; sampling must run a reverse chain.
