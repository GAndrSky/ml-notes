Быстрый старт

  1. Создай новый публичный репозиторий на GitHub.
  2. Загрузи в него index.html и остальные HTML-файлы.
  3. Открой Settings → Pages.
  4. В разделе Build and deployment выбери Deploy from a branch.
  5. Выбери ветку main и папку / (root).
  6. Сохрани настройки и дождись публикации сайта.
Сделано автоматически для публикации через GitHub Pages.
